hexo 安装

steps

  1. 必须有的环境： node, mpm, hexo
  2. mkdir demo
  3. cd demo, hexo init
  4. npm install
  5. 在source>_posts下写自己的内容，md文件
  6. hexo g 编译md -> html
  7. hexo d 发布
  8. 参考

安装hexo期间的一些问题

hexo 版本为3.0后，hexo d 命令下可能出现 ERROR Deployer not found: git
解决办法：npm install hexo-deployer-git —save
